# PayFast ITN Validation Notes

The Build20 helper validates:

- Merchant ID matches configured account.
- Signature matches local calculated signature.
- Gross amount matches checkout amount.
- Payment status is COMPLETE.

Full live launch still requires testing against PayFast sandbox/live merchant settings and externally reachable notify_url.
